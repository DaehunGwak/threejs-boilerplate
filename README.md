# webpack-threejs-boilerplate

webpack + babel + threejs

## Requirements

- node >= v12.19.0
- npm >= 7.6.3

## how to run

### install packages

```sh
npm install
# or
npm ci
```

### build

```sh
npm run build
```

### run on local

- I usually use `vscode` IDE + `live server` extension
- You can run `index.html` to use `live server` extension
